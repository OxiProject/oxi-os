## About

OxiOS is a minimal and stable Linux distribution based on Debian.

The goal of OxiOS is to provide a simple and powerful operating system with modern tools and a clean desktop experience.

## Core Components

- Oxi Package Manager
- Oxi Driver Manager
- Oxi Updater
- Oxi Game Manager

## Philosophy

- Fast
- Minimal
- Stable
- Simple
